HTML and CSS Questions and Answers

HTML Questions

What is the purpose of the <head> section in an HTML file?

The <head> section contains metadata about the HTML document that isn't displayed on the page itself. It includes information like the document title, character encoding, linked stylesheets, scripts, SEO metadata, favicon references, and other important information that helps browsers properly render the page and search engines understand its content.

How do you create a hyperlink in HTML? Provide an example.

You create hyperlinks using the <a> (anchor) tag with the href attribute: 

<a href="https://www.example.com">Visit Example Website</a>
Which HTML tag is used to insert an image into a web page?

The <img> tag is used to insert images: 

<img src="image.jpg" alt="Description of image">
What's the difference between <div> and <span>?

  • <div> is a block-level element that creates a division or section in the document. It starts on a new line and takes up the full width available.
  • <span> is an inline element used to group inline elements or apply styles to specific parts of text without breaking the flow of content.

How can you create an unordered list and an ordered list in HTML?

Unordered list (with bullet points):

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Ordered list (with numbers):

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>
How do you open a link in a new tab using HTML?

Add the target="_blank" attribute to the anchor tag: 

<a href="https://www.example.com" target="_blank">Open in new tab</a>
How do you create a table in HTML? List the essential tags used.

Essential HTML table tags:

<table>         <!-- Defines the table -->
  <thead>       <!-- Groups header content -->
    <tr>        <!-- Defines a table row -->
      <th>      <!-- Defines a header cell -->
    </tr>
  </thead>
  <tbody>       <!-- Groups body content -->
    <tr>        <!-- Defines a table row -->
      <td>      <!-- Defines a data cell -->
    </tr>
  </tbody>
</table>
What is the purpose of the alt attribute in an <img> tag?

The alt attribute provides alternative text for an image if it can't be displayed. It's essential for:

  • Accessibility for screen readers
  • SEO purposes
  • Displaying text if the image fails to load
  • Meeting web accessibility standards
How do you apply internal CSS styles to an HTML page?

Internal CSS is applied using the <style> tag in the <head> section of an HTML document: 

<head>
  <style>
    body {
      background-color: #f0f0f0;
      font-family: Arial, sans-serif;
    }
    h1 {
      color: blue;
    }
  </style>
</head>

CSS Questions

What does CSS stand for, and what is its main purpose in web design?

CSS stands for Cascading Style Sheets. Its main purpose is to separate the presentation and styling of web content from its structure (HTML). CSS allows designers to control the layout, colors, fonts, spacing, and visual appearance of web pages, providing consistency across a website and making it responsive to different devices.

How do you apply CSS to an HTML element using an internal <style> tag?
<head>
  <style>
    selector {
      property: value;
    }
    /* Example */
    p {
      color: red;
      font-size: 16px;
    }
  </style>
</head>

    <head>
  <style>
